# Презентация проекта "Мультибанк"

## Слайд 1: Титульный слайд
**Мультибанк**
Единый интерфейс финансового сервиса

VTB API Hackathon 2025

---

## Слайд 2: Проблематика
**Проблема: Фрагментация финансовых данных**

- Пользователи используют десятки отдельных приложений
- Нет единой картины финансов
- Сложно принимать финансовые решения
- Неудобное управление личными финансами

---

## Слайд 3: Решение
**Мультибанк - единая платформа**

Объединяем все банковские счета в одном интерфейсе:
- Единый просмотр всех счетов
- Агрегированная аналитика
- Переводы между счетами
- Персонализированные рекомендации

---

## Слайд 4: Ключевые возможности
**Функционал**

✅ Подключение счетов из разных банков (OAuth 2.0)
✅ Единый дашборд с балансами
✅ Аналитика доходов и расходов
✅ Переводы между счетами
✅ Экспорт данных (CSV, JSON, XLSX)
✅ Адаптивный дизайн для мобильных

---

## Слайд 5: Технологии
**Технологический стек**

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Визуализация**: Recharts
- **API**: Next.js API Routes
- **Аутентификация**: OAuth 2.0 / OpenID Connect
- **Архитектура**: Модульная, масштабируемая

---

## Слайд 6: Архитектура
**Масштабируемая архитектура**

```
Frontend (Next.js)
    ↓
API Routes (Next.js)
    ↓
Банковские API (OAuth 2.0)
├── VTB
├── Сбербанк
├── Альфа-Банк
└── Другие банки
```

Легкое добавление новых банков через единый интерфейс

---

## Слайд 7: Бизнес-модель
**Монетизация**

1. **Freemium**
   - Бесплатно: до 3 счетов, базовая аналитика
   - Премиум (₽299/мес): неограниченные счета, расширенная аналитика

2. **Партнерства**
   - Рекомендации финансовых продуктов
   - Комиссии от банков

3. **B2B**
   - Корпоративные решения
   - API для интеграций
