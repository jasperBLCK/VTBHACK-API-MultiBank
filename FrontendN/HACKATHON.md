# Мультибанк - Решение для VTB API Hackathon 2025

## Описание проекта

**Мультибанк** — единый интерфейс финансового сервиса, который позволяет пользователям управлять финансами из разных банков в одном приложении.

### Проблематика

Современные пользователи вынуждены использовать множество отдельных приложений для управления финансами:
- Один банк для зарплаты
- Другой для инвестиций
- Третий для кредитов
- Четвертый для платежей

Это приводит к фрагментации финансовой картины и неэффективному принятию решений.

### Решение

Мультибанк объединяет все банковские счета в едином интерфейсе, предоставляя:
- Единую картину всех активов и обязательств
- Инструменты для анализа и планирования
- Удобные переводы между счетами
- Персонализированную аналитику

## Бизнес-модель

### Монетизация

1. **Freemium модель**
   - Бесплатно: до 3 счетов, базовая аналитика
   - Премиум (₽299/мес): неограниченное количество счетов, расширенная аналитика, экспорт данных, приоритетная поддержка

2. **Партнерские продукты**
   - Рекомендации вкладов с комиссией от банков
   - Рекомендации кредитных продуктов
   - Инвестиционные предложения

3. **B2B сегмент**
   - Корпоративные решения для малого бизнеса
   - API для интеграции с другими сервисами

### Масштабирование

- Поддержка всех банков с открытыми API
- Локализация для других стран
- Интеграция с платежными системами
- Расширение аналитики (ML-категоризация, прогнозирование)

## Техническая архитектура

### Технологический стек

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Визуализация**: Recharts
- **API**: Next.js API Routes
- **Аутентификация**: OAuth 2.0
- **Хранение данных**: Локальное состояние + API банков

### Архитектура

```
┌─────────────────┐
│   Frontend      │
│   (Next.js)     │
└────────┬────────┘
         │
         │ OAuth 2.0
         │
┌────────▼────────┐
│  API Routes     │
│  (Next.js API)  │
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
┌───▼───┐ ┌──▼────┐
│  VTB  │ │ Other │
│  API  │ │ Banks │
└───────┘ └───────┘
```

### Масштабируемость

- Модульная архитектура банковских API
- Единый интерфейс для всех банков
- Легкое добавление новых банков через конфигурацию
- Обработка ошибок и отказоустойчивость

## Демонстрация

### Основные функции

1. **Подключение банков**
   - OAuth авторизация
   - Поддержка множественных банков

2. **Дашборд**

   - Графики доходов и расходов
   - Категоризация транзакций
   - Тренды и прогнозы

4. **Переводы**
   - Переводы между счетами
   - Расчет комиссий
   - История операций

5. **Экспорт данных**
   - Экспорт в CSV, JSON, XLSX
   - Фильтрация по датам
   - Выбор счетов
